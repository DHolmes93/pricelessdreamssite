# Run locally on your computer

## Quick start

```bash
cd pricelessdreamsite
chmod +x start-local.sh
./start-local.sh
```

Or manually:

```bash
npm install
npm run web
```

Open **http://localhost:8081** in your browser.

## Pages

- Home — http://localhost:8081/
- Products — http://localhost:8081/products
- Contact — http://localhost:8081/contact

## Requirements

- Node.js 18+ and npm
